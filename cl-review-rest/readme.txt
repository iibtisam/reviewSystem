============
Installation
============

- Install Node.js & npm
	https://nodejs.org/en/download/

- Install nodemon 
	npm install -g nodemon

- Install orm
	npm install orm

- Install orm-timestamps
	npm install orm-timestamps
	
- Install mysql
	npm install mysql

- Install express
	npm install express

- Install basic-auth
	npm install basic-auth

-Install nodemailer
	npm install nodemailer

-Install body-parser
	npm install body-parser


===============
Run Application
===============

start server
	cd cl-review-rest
	nodemon server.js

start client
	cd cl-review-web
	nodemon client.js


stop server
	for ctrl+c
